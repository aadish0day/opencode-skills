# Copy Frameworks

Reusable structures for headlines, section openers, and full landing-page arguments. Use these when you're stuck or want a faster first draft.

## Headline formulas

- {Achieve outcome} without {pain point}
  - "Close deals without the busywork"
- The {category} for {audience}
  - "The project tracker for hardware teams"
- Never {unpleasant event} again
  - "Never miss a renewal again"
- {Question highlighting main pain point}
  - "Tired of chasing approvals?"
- Ship {result} in {time}
  - "Ship your first report in 10 minutes"

## Subheadline patterns

- Clarify how: "{Headline promise} — powered by {mechanism}."
- Add specificity: "{Outcome} for {audience} who {constraint}."
- Who + what: "Built for {role}. {One concrete benefit}."

## Section templates

- Problem → Agitate → Solve
- Feature → Benefit → Proof
- Before → After → Bridge
- Claim → Example → Outcome

## Full-page argument order

1. Hero (outcome + audience + CTA + proof)
2. Problem (show you understand)
3. Solution (outcome-driven benefits)
4. How it works (3 steps)
5. Proof (testimonials / numbers / logos)
6. Objection handling (FAQ + risk reversal)
7. Final CTA
